Web de prueba

Bienvenido a esta página web de ejemplo creada para practicar el lenguaje de marcas. En este espacio se muestra contenido básico estructurado en párrafos y secciones, con el objetivo de aprender a organizar información dentro de un documento HTML. El lenguaje de marcas permite definir la estructura de una página web mediante etiquetas, facilitando la presentación ordenada del contenido. Este tipo de lenguaje es fundamental en el desarrollo web y sirve como base para crear sitios más complejos y funcionales. Este texto es únicamente de ejemplo y se utiliza para completar el diseño y la práctica de estructuras HTML.
